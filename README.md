# Hacker News MERN App

A full-stack MERN application that scrapes top stories from Hacker News, allows user authentication, and supports bookmarking functionality.

---

## Features

### Backend
- JWT Authentication (Register/Login)
- Hacker News scraper (top 10 stories)
- REST APIs for stories
- Bookmark system (toggle)
- MongoDB integration

### Frontend
- React (Vite)
- Login & Register pages
- Home page showing stories
- Bookmark functionality
- Protected routes
- React Context API for auth

---

## Tech Stack

- MongoDB
- Express.js
- React.js
- Node.js
- JWT Authentication
- Axios
- Cheerio (web scraping)
